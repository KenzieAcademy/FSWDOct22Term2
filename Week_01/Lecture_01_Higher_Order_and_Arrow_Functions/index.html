• Higher Order Function - a function that accepts another function as an argument, and executes it as a callback within its own execution
  • Callback Function - a function that is defined and provided as its definition, in order to be called back on at a later time. They are called by another function
  • Anonymous Function - a function that is defined, but not declared and saved in memory as a reusable function. Basically, a function without a name that is dealt with immediately.
  • Arrow Function - a function that defined through a special symbol (a fat arrow; =>) that functions like a normal function, but is syntactically different. It also differs in its context.
